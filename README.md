# Turtle Crossing Game

간단한 터틀 크로싱 게임입니다.  
플레이어 거북이를 위쪽으로 이동시켜 도착선에 도달하면 레벨업!  
차량은 화면 오른쪽에서 생성되어 왼쪽으로 이동하며, 충돌 시 게임 종료.

## 클래스 구조

- `CarManager`: 자동차 생성, 이동, 레벨업 속도 증가 담당
- `Player`: 거북이 주인공, 키보드 이동 및 도착선 체크
- `Scoreboard`: 레벨 표시 및 게임 오버 메시지 출력

## 사용 기술

- Python `turtle` 모듈
- 클래스와 객체지향 설계
- 랜덤 생성과 게임 루프
